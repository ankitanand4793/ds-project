this project consists of a dataset which contains information about various features which decides whether a customer using services of a telecom company will churn or not.

The graphs attached shows the various features and the number of customers who will churn based on that particulat feature. There are categorical features and numerical features for which there exists the number of customers churning.It is displayed using graph.
 
 Finally naive bayes is applied to the dataset after series of preprocessing techniques applied.
 
 The various performance metrics are displayed as the result(accuracy precision recall auc) and the accuracy graph is also displayed.
